The challenges are in the folder 'chal'.
题目放置于 'chal' 文件夹中.

Run command 'rknait' to type the flag you got, and to get the code of Alipay red envelope after you solve all the challenges.
运行指令 'rknait' 来输入你获得的 flag 并在解出所有题目后获得支付宝红包代码.

The flags you solve will be stored in the file 'flags', it is recommended to mount the file outside the container.
你所解得的 flag 将会存储在 'flags' 文件中, 建议将该文件挂载到容器外.

For detailed usage of 'rknait', see 'rknait-usage.txt'.
'rknait' 具体用法见 'rknait-usage.txt'.

All essential tools are installed for you.
所有必要的工具已经为你安装好.

注意: 所有 flag 的格式都是 'flag{xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx}' (8-4-4-4-12, uuid-like)
